### 📝 How to Read the Table's Markup

To create the table above, the following raw Markdown was used:

```markdown
| S. No. | Formatting | Markdown Syntax | Rendered Output (Example) |
| :---: | :--- | :--- | :--- |
| 1 | **Heading 1** | `# Heading 1` | (Large, main title) |
| 2 | **Heading 2** | `## Heading 2` | (Smaller subtitle) |
| ... | ... | ... | ... |
```

  * The **vertical bars (`|`)** separate the columns.
  * The **separator line** uses **dashes (`-`)** and **colons (`:`)** to define column alignment:
      * `:---:` for **center** alignment (used for S. No.)
      * `:---` for **left** alignment (default, used for most columns)
      * `---:` for **right** alignment
